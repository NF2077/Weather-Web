# 🌤️ Weather App

Sebuah aplikasi web berbasis modern yang berfungsi untuk memantau prakiraan cuaca di berbagai kota secara *real-time*. Proyek ini dibuat menggunakan HTML, CSS, JavaScript, dan mengintegrasikan [Sebutkan API-nya, misal: OpenWeatherMap API].

![Screen Shot Website Cuaca]([Link URL gambar screenshot website kamu jika ada])

## 🚀 Fitur Utama

* **Pencarian Kota Real-Time:** Cari informasi cuaca dari kota mana saja di seluruh dunia.
* **Deteksi Lokasi Otomatis:** Menampilkan cuaca di lokasi pengguna saat ini (menggunakan Geolocation API).
* **Informasi Detail:** Menampilkan suhu (°C/°F), kelembapan, kecepatan angin, indeks UV, dan tekanan udara.
* **Mode Gelap/Terang (Optional):** Kenyamanan visual saat digunakan di siang atau malam hari.
* **Desain Responsif:** Tampilan yang optimal baik diakses melalui smartphone, tablet, maupun desktop.

## 🛠️ Teknologi yang Digunakan

* **Frontend:** HTML5, CSS3, JavaScript (ES6+) / [Sebutkan framework jika pakai React/Vue/Tailwind]
* **API Penyedia Data:** [OpenWeatherMap API / WeatherAPI]
* **Icons:** [FontAwesome / Lucide Icons / Weather Icons]
